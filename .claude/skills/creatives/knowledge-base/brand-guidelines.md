# Sprinto Brand Guidelines (v1.0, Updated 15.02.2026)

## Core Colors

### Primary
- **Mulberry** `#650A41` — dominant brand color, use for backgrounds, logo, key elements
- **Texas** `#DDF07E` — primary accent, yellow-green, use for highlights and CTAs on dark backgrounds

### Secondary
- **Mulberry Wood** `#1F0214` — very dark purple, use for dark backgrounds
- **Grey** `#F5F4F5` — light backgrounds
- **Linen** `#F1C2B0` — warm peach accent
- **Melrose** `#BEB4FD` — lavender accent
- **Sail** `#B4CEFA` — light blue accent
- **Texas Light** `#EDFBA8` — light yellow-green

### Color Hierarchy
- **Dominant (50%):** `#FFFFFF`, `#F6F5F2`, `#FBEEE9`, `#650A41`
- **Supporting (10-15%):** `#E5EEFF`, `#E6E2FD`, `#F8E5DE`
- **Accent (10%):** `#6E5ADD`, `#3368C0`, `#C4633E`, `#D5EA6C`

### Color Combination Examples
- **Stats/Metrics:** `#650A41` + `#752957` + `#EDFBA8` + `#F6F5F2`
- **AI/Platform:** `#650A41` + `#5D093C` + light blues + `#D5EA6C`
- **Architecture:** `#FCF6F4` + `#F0F5FF` + `#F2F0FE` + `#FBEEE9`

## Typography

- **Primary:** Instrument Sans — use for ALL text (headlines, subheads, body, CTAs)
  - Weights: Regular, Medium, Semibold
- **Accent:** Lora — use ONLY to highlight specific words within headlines, upright only, no italic
- **Fallback:** Arial (email/system only)

### Typography Hierarchy
- Heading: Lora (accent word) + Instrument Sans, 48px
- Subtext: Instrument Sans, 16px
- Tag line: Instrument Sans, 12-18px
- CTA button: Instrument Sans, 18px

## Logo
- Mark: "S" in rounded square + "SPRINTO" wordmark
- Colors: White on Mulberry, Mulberry on Texas, Mulberry on white
- Placement: Top-left or top-right corner ONLY
- Minimum clearance: 0.5x logo height on all sides
- Never: rotate, add stroke, stretch, add gradients, add shadows, change colors

## Design Philosophy
- **Minimal, modern, bold, sleek** — neo-tech aesthetic
- No gradients, no shadows, no complex effects
- Clean geometric shapes only
- Flat and clean style
- Illustrations: product-led, UI components, system flows — never abstract or decorative

## Creative Themes
- **Light theme:** Off-white/cream (`#F6F5F2`) background + Mulberry accents
- **Dark theme:** Dark (`#1F0214` or `#1a1a2e`) background + Texas yellow-green accents
- **Thematic:** Idea-led, can break guidelines — must feel relatable and emotive

## Ad Creative Rules (Paid)
- Logo top-left corner
- CTA button: Mulberry `#650A41` background + white text OR Texas `#DDF07E` + dark text
- No gradients, no drop shadows
- Text must be legible — high contrast always
- Instrument Sans for all ad copy
- Background: `#F6F5F2` (light) or `#1F0214` (dark)
